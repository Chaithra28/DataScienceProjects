Prerequisites:
1. Latest python version installed on the system.
2. Conda Navigator through which Jupyter can be accessed.
3. You should have the training and test data sets from miner.

Follow the steps given below to execute the code:
1. Open the .ipynb file on Jupyter notebook.
2. Run the cells one by one till you see the step where the logistic_regression is called. Here you can change the number of iterations, tolerance and learning rate as required and continue with the rest of the steps.
3.The last step will save the test prediction file for both gradient descent and sklearn model. Make change to the file name as per your requirement. Upload the test prediction file on miner to check the accuracy.
